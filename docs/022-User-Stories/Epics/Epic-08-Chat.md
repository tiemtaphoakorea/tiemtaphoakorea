# Epic: Real-time Chat

## Description

Enable direct real-time communication between Admins and Customers via the platform.

## Related Specs

- [[Spec-Chat-System]]

## User Stories

- **Story-8.1**: As a Developer, I want to implement Supabase Realtime subscriptions for messaging.
- **Story-8.2**: As an Admin, I want to view a list of active conversations.
- **Story-8.3**: As a Customer, I want to chat with support from the website.
- **Story-8.4**: As a User, I want to send and receive messages instantly.
- **Story-8.5**: As a User, I want to upload and send images in the chat.
- **Story-8.6**: As a User, I want to see unread message counts.
