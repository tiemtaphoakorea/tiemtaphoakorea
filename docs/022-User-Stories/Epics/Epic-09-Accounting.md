# Epic: Accounting

## Description

Internal accounting tools to track costs, expenses, and verify financial health.

## Related Specs

- [[Spec-Finance-Accounting]]

## User Stories

- **Story-9.1**: As an Admin, I want to manage Cost Prices (Entry Price) for every variant.
- **Story-9.2**: As an Admin, I want to record Fixed and Variable expenses.
- **Story-9.3**: As an Admin, I want to manage and reconcile Debts (Công nợ).
- **Story-9.4**: As an Admin, I want to reconcile Revenue against actual payments.
- **Story-9.5**: As an Admin, I want to reconcile Inventory counts.
