# Epic: Dashboard & Reports

## Description

Visual insights and reports to help the Owner understand business performance.

## Related Specs

- [[Spec-Dashboard-Reports]]

## User Stories

- **Story-10.1**: As an Owner, I want to see charts of Revenue and Profit over time.
- **Story-10.2**: As an Owner, I want to generate a P&L (Profit & Loss) report.
- **Story-10.3**: As an Owner, I want to see Sales Reports by Product.
- **Story-10.4**: As an Owner, I want to compare performance between time periods.
- **Story-10.5**: As an Admin, I want to see suggestions for items that need re-ordering from suppliers.
