# Epic: Product Management (Admin)

## Description

Provide tools for Admins to manage the product catalog, including categories, variants, inventory, and pricing.

## Related Specs

- [[Spec-Product-Management]]

## User Stories

- **Story-3.1**: As an Admin, I want to CRUD Categories to organize products.
- **Story-3.2**: As an Admin, I want to CRUD Products with basic info and weight.
- **Story-3.3**: As an Admin, I want to manage Product Variants (Size, Color) and Options.
- **Story-3.4**: As an Admin, I want to upload product images so customers can see them.
- **Story-3.5**: As an Admin, I want to manage inventory levels for each variant.
- **Story-3.6**: As an Admin, I want to set Cost Price (Entry) and Retail/Wholesale prices per variant.
- **Story-3.7**: As an Admin, I want to filter and search products to find them quickly.
- **Story-3.8**: As an Admin, I want to set discount rules and wholesale price tiers.
- **Story-3.9**: As an Admin, I want to CRUD Suppliers so I can track product sources.
