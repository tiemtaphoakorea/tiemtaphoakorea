# Epic: Customer Page (View Only)

## Description

A read-only interface for Customers to browse the product catalog and view details.

## Related Specs

- [[Spec-Customer-Catalog]]

## User Stories

- **Story-4.1**: As a Customer, I want to view a catalog of products.
- **Story-4.2**: As a Customer, I want to see product details (Price, Description, Stock Status).
- **Story-4.3**: As a Customer, I want to search and filter products by category or name.
