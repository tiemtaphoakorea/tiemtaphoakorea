## Mô tả

Nhân viên xem danh sách sản phẩm, chỉnh sửa thông tin cơ bản và cập nhật tồn kho từng biến thể.

<Warning>
Hạn chế quyền nhân viên:
- **Không** được tải lên / xoá ảnh sản phẩm.
- **Không** được xoá biến thể đã có lịch sử bán hàng.

Liên hệ Quản lý / Chủ shop khi cần các thao tác trên.
</Warning>

## Cách truy cập

Menu bên trái → **Sản phẩm**.

## Trang danh sách

### Bộ lọc nhanh (tab)

| Tab | Lọc theo |
|-----|----------|
| **Tất cả** | Mọi sản phẩm |
| **Còn hàng** | Tồn > ngưỡng |
| **Sắp hết** | 0 < Tồn ≤ ngưỡng |
| **Hết hàng** | Tồn = 0 |

### Tìm kiếm

Ô **Tìm tên, mã SP...** — debounce 300ms.

### Cột bảng

| Cột | Nội dung |
|-----|---------|
| **Sản phẩm** | Thumbnail + tên + SKU |
| **Danh mục** | Danh mục cha |
| **Giá bán** | Giá hoặc khoảng giá |
| **Tồn kho** | Tổng tồn (đỏ nếu = 0, vàng nếu thấp) |
| **Trạng thái** | Đang bán / Hết hàng / Ẩn |
| **Sửa** | Mở drawer chỉnh sửa |

### Phân trang

Cuối bảng — dropdown **10 / 20 / 50 / 100**. Mặc định **20**.

## Các thao tác chính

<Steps>
  <Step title="Tìm sản phẩm">
    Dùng ô tìm hoặc tab lọc để tìm sản phẩm cần cập nhật.
  </Step>
  <Step title="Mở chi tiết">
    Nhấn vào hàng → drawer chỉnh sửa mở.
  </Step>
  <Step title="Cập nhật tồn kho">
    Tab **Biến thể & Giá** → chỉnh số lượng tồn kho từng biến thể trong bảng.
  </Step>
  <Step title="Cập nhật thông tin cơ bản">
    Tab **Thông tin chung** → đổi tên, mô tả, danh mục, giá niêm yết.
  </Step>
  <Step title="Đánh dấu nổi bật">
    Tab **Thông tin chung** → switch **Bán chạy / Nổi bật** sang `ON` → **Lưu**.
  </Step>
</Steps>

## Trường có thể chỉnh

| Trường | Mô tả |
|--------|------|
| **Tên sản phẩm** | Tên hiển thị trên storefront |
| **Danh mục** | Cây danh mục |
| **Giá niêm yết** | Giá mặc định cho biến thể tạo mới |
| **Mô tả** | Mô tả chi tiết |
| **Hiển thị** | Switch ẩn/hiện trên storefront |
| **Bán chạy / Nổi bật** | Switch — đưa lên trang chủ |
| **Biến thể: Giá bán / Giá vốn / Tồn kho / Ngưỡng cảnh báo** | Trong bảng tab Biến thể |

<Note>
Khi nhập sai tồn kho, có thể dùng **Đơn nhập hàng** hoặc **Kiểm kê tồn kho** để điều chỉnh chính xác.
</Note>
