## Mô tả

Dashboard hiển thị các chỉ số vận hành quan trọng: đơn hàng cần xử lý và sản phẩm sắp hết hàng.

## Cách truy cập

Menu bên trái → **Dashboard**.

## Các thao tác chính

<Steps>
  <Step title="Kiểm tra đơn hàng mới">
    Xem widget đơn hàng mới để biết có đơn nào cần xác nhận thanh toán hoặc cập nhật trạng thái.
  </Step>
  <Step title="Kiểm tra tồn kho thấp">
    Widget sản phẩm sắp hết hàng cảnh báo các sản phẩm cần nhập thêm — báo lại cho Quản lý hoặc Chủ shop nếu cần tạo đơn nhập.
  </Step>
</Steps>

<Note>
Dữ liệu Dashboard cập nhật theo thời gian thực. Tải lại trang nếu cần xem số liệu mới nhất.
</Note>
