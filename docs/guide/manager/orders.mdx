## Mô tả

Trang Đơn hàng là trung tâm xử lý tất cả đơn hàng: xác nhận thanh toán và cập nhật trạng thái giao hàng.

## Cách truy cập

Menu bên trái → **Đơn hàng**.

## Vòng đời đơn hàng

```
Chờ xác nhận → Đã thanh toán → Đang chuẩn bị → Đang giao → Đã giao
```

## Các thao tác chính

<Steps>
  <Step title="Lọc đơn hàng cần xử lý">
    Dùng bộ lọc trạng thái để hiển thị đơn hàng **Chờ xác nhận** cần được xử lý.
  </Step>
  <Step title="Xác nhận thanh toán">
    Mở đơn hàng → nhấn **Xác nhận thanh toán** → chọn phương thức (tiền mặt/chuyển khoản/thẻ) → nhập số tiền nhận được.
  </Step>
  <Step title="Cập nhật trạng thái giao hàng">
    Nhấn nút chuyển trạng thái: **Đang chuẩn bị** → **Đang giao** → **Đã giao**.
  </Step>
</Steps>

<Note>
Đơn hàng hỗ trợ thanh toán một phần. Nhập đúng số tiền khách đã trả — phần còn lại sẽ ghi nhận là nợ.
</Note>
