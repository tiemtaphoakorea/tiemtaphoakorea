# TC-ACC-008: Daily Report Totals

## Pre-conditions

- Logged in as Admin.
- Delivered orders exist across multiple days.

## Test Steps

1. Open the report for a date range (e.g., last 7 days).
2. Compare daily chart totals with delivered order sums per day.

## Expected Result

- Daily revenue, cost, and profit match delivered orders for each date.
