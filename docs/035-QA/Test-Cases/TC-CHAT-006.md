# TC-CHAT-006: Mark Messages as Read

## Pre-conditions

- Logged in as Admin.
- Chat room has unread customer messages.

## Test Steps

1. Open the chat room.
2. Observe unread badge count.

## Expected Result

- Unread messages are marked as read.
- Unread badge count decreases to 0 for that room.
