# TC-ACC-010: Expense Edit & Delete

## Pre-conditions

- Logged in as Admin (Owner).
- An expense record exists.

## Test Steps

1. Edit the expense amount and save.
2. Delete the expense.
3. Open P&L report.

## Expected Result

- Edited expense updates in list and totals.
- Deleted expense is removed and totals update accordingly.
