# TC-DASH-010: Dashboard Date Range Filter

## Pre-conditions

- Logged in as Admin or Owner.
- Orders exist across multiple days.

## Test Steps

1. Open Dashboard and select a custom date range.
2. Note KPI values and charts.
3. Change to a different date range.

## Expected Result

- KPIs and charts update to reflect the selected range.
- Values match expected totals for each range.
