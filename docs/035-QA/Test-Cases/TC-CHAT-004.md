# TC-CHAT-004: Real-time Text Messaging

## Pre-conditions

- Admin and customer are in the same chat room.

## Test Steps

1. Customer sends a text message.
2. Admin receives the message without refresh.
3. Admin replies with a text message.

## Expected Result

- Both sides receive messages in real-time.
- Messages show timestamps and sender identity.
